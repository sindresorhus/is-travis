# is-travis

> Check if your code is running on [Travis](https://travis-ci.org)


## Install

```
$ npm install is-travis
```


## Usage

```js
const isTravis = require('is-travis');

// On your local computer
console.log(isTravis);
//=> false

// On Travis
console.log(isTravis);
//=> true
```
